# 📊 Telecom Data Science & Machine Learning Pipeline

## 📌 Project Overview
This repository demonstrates a complete data science workflow inspired by tasks performed during my internship at **Zain Jordan**.  
To respect corporate confidentiality, the dataset used here is synthetic/public, but the pipeline reflects real-world industry practices.

The project covers data preprocessing, exploratory analysis, machine learning model development, and hyperparameter optimization.

---

## 🔧 Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Jupyter Notebook  

---

## 📂 Project Structure

notebooks/

02_exploratory_data_analysis.ipynb

---

## 🧠 Workflow Description

### 1️⃣ Data Cleaning and Preprocessing  
**01_data_cleaning_and_preprocessing.ipynb**
- Data loading and inspection  
- Handling missing values  
- Encoding categorical variables  
- Feature scaling and preparation for modeling  

---

### 2️⃣ Exploratory Data Analysis (EDA)  
**02_exploratory_data_analysis.ipynb**
- Statistical summaries  
- Distribution and correlation analysis  
- Visualizations to understand feature relationships and patterns  

---

### 3️⃣ Model Training and Evaluation  
**03_model_training_and_evaluation.ipynb**
- Train-test split  
- Baseline machine learning models  
- Performance evaluation using accuracy, precision, recall, and F1-score  

---

### 4️⃣ Model Training and Hyperparameter Tuning  
**04_model_training_and_hyperparameter_tuning.ipynb**
- Training multiple ML models (Logistic Regression, Random Forest, SVM, KNN, Gradient Boosting)  
- Cross-validation using StratifiedKFold  
- Hyperparameter optimization with RandomizedSearchCV  
- Evaluation using ROC-AUC and classification metrics  

---

## 📈 Key Outcomes
- Built a reproducible end-to-end ML pipeline  
- Compared multiple models and evaluation strategies  
- Demonstrated industry-standard ML validation techniques  
- Produced interpretable visualizations and insights
